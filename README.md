## Install

```bash
git clone https://github.com/Fudenberg-Research-Group/akita_utils.git
cd akita_utils
make install
```
## Usage

```py
from akita_utils import BaseClass
from akita_utils import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m akita_utils
#or
$ akita_utils
```
